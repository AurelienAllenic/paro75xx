- Start with the documentation.

- Please note, youtube background, google maps, popups and forms must be deployed on a web server or local server environment ( MAMP, XAMPP, LAMP, ... ) to function. 
  If you load the the html files directly into your browser the youtube background, google map, popups and form will may not work correctly.

- You need a Google Maps API key to use the Google map. [ https://developers.google.com/maps/documentation/javascript/get-api-key ] Insert it in your config.js file under section "2. MAIN CONFIG".

- You can change the background and other options via config.js file.
